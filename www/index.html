AutoIt HTTP Web Server

To add PHP functionality:
  • Download your desired version from here
  • Create a new folder in the same directory as the server script
  • Extract the archive to the newly created folder
  • Open settings.ini and set the PHP_Path value to the folder name
  • Optional: If you want to allow or disallow indexing if a directory is requested and no index file is found:
    • Open settings.ini and set AllowIndexes to True or False, depending on your need.
  • (Re-)Start the server
